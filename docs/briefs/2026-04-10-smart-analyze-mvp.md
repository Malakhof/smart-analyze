# Feature Brief: Smart Analyze — AI Sales Analytics Platform

> **Date:** 2026-04-10
> **Source:** Adaptive interview (4 questions)

## Goal

Клон продукта "Умный Анализ" (Obuchat AI) как база + собственные улучшения сверху.
SaaS-платформа для AI-аналитики отделов продаж: забирает данные из CRM, анализирует сделки (переписки + звонки), находит паттерны успеха/провала, рейтингует менеджеров, даёт конкретные рекомендации с цитатами.

## Target User

- **Руководители отделов продаж (РОПы)** — хотят видеть кто из менеджеров работает, а кто нет
- **Собственники бизнеса** — хотят понять почему деньги теряются и как увеличить конверсию
- **Сегмент:** B2B компании с 3+ менеджерами, сотни сделок в месяц, используют Битрикс24 или amoCRM

## Success Criteria

- [ ] Рабочий SaaS с регистрацией и тарифами (Demo 50 сделок бесплатно / Базовый / Стандарт / Про)
- [ ] Интеграция Битрикс24 + amoCRM (автозагрузка сделок, воронок, переписок, звонков)
- [ ] Дашборд: воронка с конверсиями, успех/провал, потенциал роста выручки, ключевые метрики
- [ ] Рейтинг менеджеров: таблица с метриками + drill-down с AI-рекомендациями и цитатами
- [ ] Библиотека паттернов: успех vs провал с метриками (сила, влияние, надёжность, охват)
- [ ] Транскрибация звонков (WhisperX) + анализ текстовых переписок
- [ ] Деплой на Timeweb сервер (80.76.60.130)

## Scope

**In scope:**
- 5 экранов: Главная (дашборд), Менеджеры (список), Менеджер (drill-down), Библиотека паттернов, Настройки (CRM-подключение)
- Тёмная тема (как у оригинала) + светлая
- SaaS: регистрация, мультитенант, тарифные планы
- Интеграция: Битрикс24 REST API + amoCRM API (архитектура под расширение)
- AI-движок: Claude API для анализа сделок, поиска паттернов, генерации рекомендаций
- Аудио: WhisperX для транскрибации звонков
- Фильтры по периоду (день/неделя/месяц)

**Out of scope (пока):**
- AI-коуч в реалтайме
- A/B тест скриптов
- TG-бот уведомлений
- Мобильное приложение
- Биллинг/оплата (на MVP — ручное управление тарифами)

## Constraints & Preferences

- **Tech stack:** Next.js + TypeScript + shadcn/ui + Tremor + PostgreSQL + Claude API
- **Deploy:** Docker на Timeweb сервер (рядом с QUP/Soldout)
- **Дизайн:** Тёмная тема по умолчанию, как у оригинала. Минималистичный, data-heavy UI
- **Архитектура CRM:** адаптерный паттерн — единый интерфейс, плагины под каждую CRM
- **Улучшения поверх клона:** определим после запуска базовой версии
