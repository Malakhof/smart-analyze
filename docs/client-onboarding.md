## Инструкция для amoCRM (OAuth)

### Что нужно от клиента: 3 ключа + код авторизации

📋 **Шаг 1.** Откройте amoCRM → **amoМаркет** (левое меню) → **«+ Создать интеграцию»**

📋 **Шаг 2.** Заполните форму:

| Поле | Что вписать |
|------|-------------|
| Название | `SalesGuru Integration` |
| Описание | `Интеграция SalesGuru для анализа сделок и звонков` |
| URL | `https://sa.qupai.ru/hook/inbound` |
| Redirect URI | `https://sa.qupai.ru/hook/inbound` |
| Предоставить доступ | ✅ **Все** |

Нажмите **«Сохранить»**

📋 **Шаг 3.** Откройте вкладку **«Ключи и доступы»**

📋 **Шаг 4.** Скопируйте и отправьте нам **3 значения:**

```
🔑 ID интеграции:
(формат: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx)

🔑 Секретный ключ:
(длинная строка)

🔑 Код авторизации (действителен 20 минут!):
(длинная строка, начинается с def50200...)
```

⚠️ **ВАЖНО:** Код авторизации действует только **20 минут**! Копируйте его последним, прямо перед отправкой нам.

## Инструкция для Битрикс24

### Вариант 1: Входящий вебхук (проще, до 50 заявок/день)

📋 **Шаг 1.** Откройте Битрикс24 → **Настройки** → **Разработчикам** → **Другое** → **Входящий вебхук**

📋 **Шаг 2.** Создайте вебхук:

| Поле | Что вписать |
|------|-------------|
| Название | `SalesGuru Integration` |
| Права доступа | ✅ **CRM (crm)** |

Нажмите **«Сохранить»**

📋 **Шаг 3.** Скопируйте URL вебхука и отправьте нам:

```
🔑 URL вебхука:
(формат: https://ваш-домен.bitrix24.ru/rest/USER_ID/SECRET/)

Пример: https://mycrm.bitrix24.ru/rest/1/abcdef1234567890/
```

### Вариант 2: OAuth (для большого трафика)

📋 **Шаг 1.** Откройте Битрикс24 → **Настройки** → **Разработчикам** → **Другое** → **Локальное приложение**

📋 **Шаг 2.** Заполните форму:

| Поле | Что вписать |
|------|-------------|
| Тип | **Серверное** |
| Путь обработчика | `https://sa.qupai.ru/hook/inbound` |
| Путь установки | `https://sa.qupai.ru/hook/inbound` |
| Использует только API | ✅ |
| Права | ✅ **CRM (crm)** |

Нажмите **«Сохранить»**

📋 **Шаг 3.** Откройте вкладку **«Ключи и доступы»**

📋 **Шаг 4.** Скопируйте и отправьте нам:

```
🔑 Код приложения (client_id):
(формат: local.XXXXX.XXXXX)

🔑 Ключ приложения (client_secret):
(длинная строка)
```

## После получения ключей (наши действия)

1. Вводим ключи в Settings → CRM
2. Нажимаем "Синхронизировать"
3. Ждём 2-5 минут (зависит от количества сделок)
4. Запускаем AI-анализ: DeepSeek обрабатывает сделки
5. Если есть звонки: Whisper транскрибирует → DeepSeek анализирует
6. Через 24 часа клиент видит полный кабинет с данными
