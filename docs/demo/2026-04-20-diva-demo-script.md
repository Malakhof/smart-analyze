**URL:** `https://app.salezguru.ru` (после пропагации DNS) или fallback `https://sa.qupai.ru`

## Fallback: что делать если что-то не работает

| Проблема | Решение |
|---|---|
| Dashboard тормозит без VPN | Открыть через VPN, скрыть это от клиента. Объяснить: "для RU делаем прямой DNS без CF, завтра будет быстрее" |
| Transcript пустой для конкретного звонка | Кликнуть в другой — у нас 400+ |
| Insight показывает ошибку markdown | Обновить страницу |
| Quality показывает "0" где-то | Открыть другого менеджера |
