## 🎯 ЧТО МЫ СДЕЛАЕМ ПОСЛЕ ВАШИХ ОТВЕТОВ

| Ответ | Действие в системе |
|---|---|
| Главная воронка = "Горячие" | Конверсия считается только по ней, "Отказ" уходит в "отбраковка" |
| Список реальных менеджеров | Скроем 24 неактивных, "Александр" — либо удалим либо отдельно |
| Custom-поле суммы | Подтянем правильный "Бюджет" → честный средний чек |
| Категории звонков | Настроим автотеги под ваши типы (жильцы / агентства / бухгалтерия) |
| Скрипты | Заменим универсальный на ваш — оценка качества станет релевантной |
| Чек-лист ошибок | Автодетект "критических ошибок" — алерт РОПу |
| Причины отказа | Топ-3 причин потерь — ответ "почему теряем" |
| Дубли | Раздел "кандидаты на слияние" с one-click merge |
