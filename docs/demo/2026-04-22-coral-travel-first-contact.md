## Специфика туристических агентств (что ожидать)

**Бизнес-модель:**
- Горячие туры (клиент пришёл → купил сегодня-завтра)
- VIP-туры (долгое согласование, индивидуалка)
- Групповые, корпоративы
- Сезонность (летом/зимой пик, межсезонье — upsell / win-back)

**Типичные звонки:**
- Входящий по рекламе / Авито / Яндекс-директ
- Возврат постоянного клиента
- Допродажа туров (страховка, экскурсии, трансфер)
- Переоформление под санкции / отмены
- Поддержка во время тура

**Боли (гипотезы для продажи):**
- Сезонный пик → физически нельзя прослушать все звонки → AI закрывает
- Менеджеры работают по-разному → стандарт не соблюдается → AI находит отклонения
- Сливы дорогих VIP-клиентов → AI отмечает риск по тону / паузам / возражениям
- Новичок vs топ → AI показывает конкретно что топ говорит что новичок не говорит

**Типичный стек в Челябинске (Coral Travel):**
- Скорее всего **amoCRM** (стандарт для туристических сетей)
- Телефония — зависит от франчайзи (Coral — франчайзинговая сеть, каждый офис сам решает)
- Могут быть **кастомные интеграции с FranchisePlatform** (система Coral для франчайзи)
