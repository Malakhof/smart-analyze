### Что сделать в кабинете МегаПБХ — пошагово (5 минут)

**Заходите:** `https://coral-travel.megapbx.ru` (логин/пароль администратора)

**Шаг 1 — Включить стерео-запись:**
```
Настройки → Запись разговоров
  → Формат записи: выбрать "Стерео" (вместо "Моно")
  → Включить "Запись всех вызовов" (если ещё не стоит)
  → Сохранить
```

**Шаг 2 — Включить хранение записей не менее 90 дней:**
```
Настройки → Запись разговоров → Хранение
  → Период хранения: 90+ дней (или "Бессрочно" если тариф позволяет)
  → Сохранить
```

**Шаг 3 — Дать API доступ для нашей интеграции:**
```
Настройки → Интеграции → API
  → "Создать API ключ" / "Сгенерировать токен"
  → Назвать: "Sales GURU integration"
  → Права: "Чтение записей звонков" + "Чтение истории звонков" (read-only)
  → Скопировать ключ → прислать нам
```

**Шаг 4 — Включить webhook на новые звонки** (если есть в тарифе):
```
Настройки → Webhooks → Добавить
  → URL: https://app.salezguru.ru/api/webhook/megapbx/coral
  → События: "Звонок завершён", "Запись готова"
  → Сохранить
```

После этого:
- Все НОВЫЕ звонки → стерео автоматически
- Мы качаем напрямую через API (без зависимости от amoCRM)
- При webhook — обработка в real-time (через 2-5 минут после звонка)

**Старые 8 504 моно-звонка** остаются в системе для **общей статистики количества и времени** (кто сколько звонил), но в **deep-анализ скриптов не включаем** (моно = бесполезно).

⏰ **ETA до полного analytics:** 1-2 недели накопления стерео базы (≥1000 звонков) → запускаем deep-pipeline (соответствие скрипту, лучшие фразы, риски слива).
