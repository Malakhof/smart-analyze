## Текущий этап: MVP Launch

### Модель работы AI-агента (3 фазы)

**Фаза 1: Разведка (read-only, безопасно)**
- Агент логинится в CRM клиента (amoCRM / GetCourse)
- Изучает структуру: звонки, менеджеры, заказы, воронки
- Парсит ВСЕ звонки → транскрибирует через Whisper
- Парсит переписки, заказы, статусы
- Возвращает отчёт: "вот что вижу, вот сколько данных"
- **Полностью read-only — ничего не пишем в CRM клиента**

**Фаза 2: Анализ (на нашей стороне)**
- DeepSeek анализирует каждый звонок и переписку
- Выявляет паттерны успеха (что говорят менеджеры которые продают)
- Выявляет паттерны неуспеха (где теряются сделки)
- Если есть скрипт продаж — сравнивает с реальными паттернами
- Генерирует идеальный скрипт на основе данных
- Рекомендации по каждому менеджеру индивидуально
- Заполняет дашборд: воронки, этапы сделок, древо сделок, рейтинг

**Фаза 3: Согласование**
- Показываем клиенту результат на sa.qupai.ru
- Клиент подтверждает → подписка 50K₽/мес
- Дальше — ежедневный мониторинг и обновление аналитики

### Безопасность

- Фазы 1 и 2 — полностью read-only
- Все данные копируются к нам, обрабатываются у нас
- Бэкапы нашей БД перед каждым крупным действием
- Дедупликация — только отчёт "вот дубли", без удаления. Клиент сам решает
- NDA подписан до начала работы
- Доступ отзывается клиентом одним кликом

## Следующие этапы развития

### Этап 2: Мобильное приложение

**Цель:** Дистанционное управление и получение отчётов без привязки к десктопу.

**Порядок:**
1. **Android** — первый (больше аудитория в РФ среди РОПов)
2. **iOS** — следом

**Функции мобильного приложения:**
- Push-уведомления: критические звонки, падение конверсии, новые инсайты
- Дашборд с ключевыми метриками
- Рейтинг менеджеров
- Прослушивание звонков с транскрипцией
- Ежедневная сводка
- Управление подпиской

### Этап 3: Модель продвижения — Пирамидальная партнёрская программа

**Идея:** Формирование сообщества амбассадоров через профессиональную сеть.

**Целевые партнёры:**
- **HR-директора** — видят проблему текучки и обучения менеджеров
- **РОПы (руководители отделов продаж)** — прямые пользователи, знают боль
- **КОМы (коммерческие директора)** — принимают решение о бюджете

**Каналы привлечения партнёров:**
- HH.ru — поиск HR/РОП/КОМ через вакансии и резюме
- LinkedIn — профессиональная сеть
- Профильные конференции и вебинары

**Модель монетизации партнёров:**
- Партнёрский процент от каждого привлечённого клиента
- Пирамида: % от партнёров привлечённых партнёрами
- % от подписок привлечённых клиентов (рекуррентный доход)

**Результат:** База амбассадоров → профессиональное сообщество → позиционирование: "без AI-РОПа невозможно полноценно управлять отделом продаж"

## Производительность (доказано)

```
HTTP запрос к GetCourse:    ~0.5 сек (нет API лимитов)
100 звонков парсинг:        ~50 сек
Whisper + DeepSeek:         ~2 мин/звонок
100 звонков полный анализ:  ~3 часа (1 поток) / ~45 мин (4 потока)
```

## Стек

- Web: Next.js 16 + Prisma 7 + PostgreSQL
- AI: DeepSeek V3 (анализ) + WhisperX (транскрибация)
- CRM: amoCRM (OAuth API) + GetCourse (HTTP session) + Bitrix24 (webhook)
- Деплой: Docker на Timeweb, Caddy, Cloudflare
- Live: https://sa.qupai.ru
