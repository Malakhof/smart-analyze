## 2. Себестоимость обработки 1 звонка

### Whisper (транскрипция)

**OpenAI Whisper API:** $0.006/мин

| Длительность звонка | Стоимость |
|---|---|
| 2 мин (reklama avg) | $0.012 = **1₽** |
| 3.3 мин (vastu avg) | $0.020 = **1.7₽** |
| 5 мин | $0.030 = **2.5₽** |

**Faster-Whisper на vast.ai spot GPU (RTX 3060, ~8x realtime, $0.15/hr):**
- 1 час аудио = 0.125 GPU-hours × $0.15 = **$0.02/час**
- Per 3-мин звонок: **$0.001 = 0.08₽**

### DeepSeek V3 (анализ)

DeepSeek V3: $0.27/M input, $1.10/M output

Per звонок (transcript + context → summary):
- Input: ~3000 tokens (транскрипт + сделка + предыдущие сообщения)
- Output: ~800 tokens (summary + паттерны)
- Cost: 3000/1M × $0.27 + 800/1M × $1.10 = **$0.002 = 0.15₽**

### ИТОГО per звонок (transcription + analysis)

| Стек | Per звонок (2-5 мин) | Клиент reklama 119 зв/мес | Клиент vastu 1445 зв/мес |

| **OpenAI API + DeepSeek** | **~1-3₽** | **~150-350₽/мес** | **~1,800-4,300₽/мес** |

## 5. Тех-стек Whisper: сравнение альтернатив

### Стандарт индустрии для batch-транскрипции

| Решение | Скорость | VRAM | Качество | Использование |
|---|---|---|---|---|
| OpenAI Whisper API | — | — | эталон | прямой HTTP |
| **Faster-Whisper** ⭐ | 4x быстрее | 2-6 GB | = OpenAI | продакшн стандарт |
| Insanely-Fast-Whisper | 20x с Flash Attention 2 | 8-16 GB | = OpenAI | новые GPU |
| WhisperX | 4x + диаризация | 6-8 GB | = OpenAI + speaker | когда нужно "кто говорит" |
| Distil-Whisper | 6x | 2-4 GB | чуть хуже | edge cases |
| whisper.cpp | быстро на CPU | RAM only | = OpenAI | M1/M2/M3 Mac |

**Для нашего SaaS:** Faster-Whisper + pyannote диаризация (говорит кто) = Faster-Whisper v3 в комплекте или WhisperX wrapper.

### Рекомендуемое железо

| Setup | VRAM | CPU | RAM | Стоимость |
|---|---|---|---|---|
| Минимум (Faster-Whisper medium + pyannote) | 6 GB | 4 core | 16 GB | vast.ai spot ~$0.15/ч |

## 6. Фазированный rollout

### Фаза A — Launch (первые 2-3 клиента)

- **OpenAI Whisper API** + DeepSeek V3 для анализа
- Setup: 1 день (уже есть `src/lib/crm/...` — нужен только transcription worker + DeepSeek prompt)
- Себестоимость per клиент: ~150-350₽/мес (reklama), ~2,000₽/мес (vastu)
- Маржа: 95-98%
- Dev focus: продукт, UI, дашборд

### Фаза B — Scale (3-5+ клиентов или vastu-уровень)

- **Миграция на Faster-Whisper на vast.ai/runpod spot GPU**
- Setup: 1-2 дня (Docker worker + queue + retry + vast.ai API)
- Себестоимость per клиент: падает в 5-15x
- Маржа: ≥99%
- Vastu и подобные крупные клиенты перестают быть "бременем"

### Фаза C — Industrial (20+ клиентов, >6K часов/мес суммарно)

- Переход на **dedicated Cloud GPU** (Selectel / Timeweb / свой сервер)
- Предсказуемый биллинг, меньше flakiness от spot
- Cost: ~$120-300/мес fixed, но амортизируется на N клиентах

## 7. Pre-sale демо-стратегия

### Flow

1. РОП заполнил анкету → подключился (по PDF инструкции)
2. Pre-flight: показываем количество сделок/воронок в CRM → он решает что включить в демо
3. Первый sync: **50 самых свежих сделок + 5 часов звонков**
4. DeepSeek анализирует → 3-5 паттернов по ЕГО данным
5. Презентация: показываем ЕГО менеджеров, ЕГО цифры, ЕГО цитаты

**Себестоимость демо:**
- Сценарий A: ~$2 = 160₽ на лид
- Сценарий B: ~$0.30 = 25₽ на лид

Даже при 100 демо/мес = максимум 16,000₽ на привлечение. Копейки.

### Защита от abuse

- Демо живёт **14 дней**. Не купил → sync отключается, данные замораживаются в БД (для ресурекция при покупке)
- Повторное демо нельзя (кроме случая "переоткрытие после паузы" — тогда с обнулением)
- Запрос расширения "покажи мне больше" = "это уже в тарифе"

### Что показываем в демо-презентации (key hooks)

Продаём **деньги**, не софт:
- "Менеджер Александр: 0 WON из 74 сделок. Знаете об этом?"
- "Паттерн: когда цена в первые 30 мин звонка — +47% конверсия"
- "Воронка Х мертва (0 за 3 мес), занимает место в CRM"
- "10 часов транскрипций показали: 60% отказов — цена. Готовим скрипт возражений"

## 8. Killer features для удержания после продажи

### "Calendar of improvements"

Персональный план 30-дневных действий по результатам анализа:
- **Неделя 1:** прослушать 5 звонков Александра (в них паттерн X)
- **Неделя 2:** переобучить 3 менеджера под паттерн Y
- **Неделя 3:** A/B-тест скрипта возражения "дорого"
- **Неделя 4:** ретрансляция лучших звонков команде

Превращает нас из "дашборда" в **AI-РОПа**. Оправдывает 250K₽/мес.

### Weekly digest

Еженедельный email: "Ваш РОП-отчёт за неделю":
- Топ/дно менеджеров
- Сделки, которые "застряли" — кому звонить сейчас
- Новые паттерны которые AI обнаружил
- Сравнение с прошлой неделей

### "Replay mode"

В UI: кликаешь на сделку → видишь полную историю: звонки (с транскрипцией + aligned текст), сообщения, переходы этапов, задачи. Чтобы РОП/менеджер мог пересобрать кейс одним кликом.

## C. Когда переходить на свой сервер (break-even)

### Экономика своей сборки

| Компонент | Цена |
|---|---|
| RTX 3060 12GB | 40,000₽ |
| Mobo + CPU + RAM 32GB + PSU + case + SSD | 50,000₽ |

| Амортизация за 3 года | ~2,500₽/мес |
| **Monthly cost** | **~3,700₽/мес ($46)** |

### Break-even vs runpod ($0.20/час GPU)

```
$46/мес / $0.20/час = 230 часов compute в месяц
При Faster-Whisper 8x realtime: 230ч compute = 1,840ч аудио/мес
```

| Масштаб | Compute часов/мес | Runpod $/мес | Свой $/мес | Победитель |

| **50 клиентов** | ~250 | $50 | $46 | **свой** (чуть) |
| 100 клиентов | ~500 | $100 | $46 | **свой** (2.2x) |
| 1000+ клиентов | 5000+ | $1,000+ | $46 + мониторинг | **свой парк серверов** |

### Примечание про Cloud GPU dedicated (Selectel и т.п.)

Dedicated Cloud GPU ~$120/мес RTX 3060:
- Break-even vs runpod: 600 часов compute = ~60-80 клиентов mix
- Хуже самосборки до очень больших объёмов
- Имеет смысл только когда **нужна SLA гарантия** (не в подвале, а в дата-центре)

### Подводные камни своей сборки

- Перегрев → падение throughput
- Электричество могут отключить
- Интернет упал = все клиенты ждут
- Сломалась плата — 3-5 дней downtime
- Для business-grade SLA → колокейшн в ДЦ (+3,000-5,000₽/мес)
