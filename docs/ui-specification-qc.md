# Smart Analyze — Модуль "Контроль качества" (QC)

> Источник: вторая видео-презентация Obuchat AI

## Суть модуля

Помимо "Умного Анализа" (пост-фактум анализ сделок), есть **контроль качества звонков в реалтайме**:
- Менеджер разговаривает → звонок записывается → AI слушает → проверяет по скрипту → выставляет балл
- Если менеджер пропустил критичный пункт → **мгновенный алерт в Telegram РОПу**
- KPI менеджеров привязан к баллам

## Новые экраны

### QC Dashboard
- Выполнение скрипта % (по отделу, по менеджеру)
- Динамика по периодам (день/неделя/месяц)
- Разбивка по категориям звонков (входящие/исходящие/допродажа)
- Теги: кастомные метки на звонках (например "не озвучил бесплатную доставку")

### QC Менеджер (drill-down)
- Список всех диалогов менеджера
- Каждый диалог: дата, длительность, балл %, статус (норма/проблема)
- Провалиться в конкретный диалог

### QC Диалог (детальный разбор)
- **Транскрибация** полная (текстом)
- **Аудиоплеер** (послушать по-быстрому)
- **Чеклист скрипта:** список обязательных пунктов
  - ✅ Сделано / ❌ Не сделано — по каждому пункту
  - Общий балл: XX%
- **AI-комментарии** к каждому пункту

### QC Настройки скриптов
- Создание/редактирование скриптов
- Список обязательных пунктов (чеклист)
- Категории звонков (входящие, исходящие, допродажа)
- Настройка весов пунктов (влияние на общий балл)
- Привязка скрипта к категории

### Telegram-алерты
- Настройка: какие пункты критичные → алерт в TG
- Бот отправляет: "Менеджер [Имя] не сказал [пункт] в звонке #[ID] с [клиент]"
- Настройка в разделе Настройки → Уведомления

## Изменения в существующих экранах

### Хедер — новый пункт навигации
- Главная | Менеджеры | Паттерны | **Контроль качества** | Настройки

### Dashboard (главная) — новые метрики
- "Всего сделок: 52" с подписью "с 26 коммуникациями"
- Заголовок страницы: "Дашборд отдела продаж"
- Фильтр "Все время" добавить к периодам
- Поиск в хедере

### Настройки — новые разделы
- Скрипты (создание чеклистов)
- Telegram-бот (подключение, настройка алертов)

## Ценообразование (обновлённое)
- Умный Анализ: 100₽ за сделку
- Контроль качества: 150₽ интеграция + 3-5₽/минута
- Переписки: 25-30₽ за диалог
- Партнёрка: 20%

## Новые модели данных

```
Script — скрипт (название, категория, привязка к tenant)
ScriptItem — пункт скрипта (текст, вес, isCritical для TG-алертов)
CallRecord — запись звонка (менеджер, клиент, аудио URL, длительность, транскрипт)
CallScore — оценка звонка (общий балл %, привязка к скрипту)
CallScoreItem — оценка по пункту (сделано/не сделано, AI-комментарий)
CallCategory — категория (входящий/исходящий/допродажа, кастомные)
CallTag — тег на звонке
TelegramConfig — настройки TG-бота (chat_id, token, какие пункты алертить)
```
