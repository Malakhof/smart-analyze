# Smart Analyze — Полная UI-спецификация

> Источник: скриншоты + расшифровка видео-презентации Obuchat AI "Умный Анализ"

---

## Экран 1: Главная (Dashboard)

### 1.1 Хедер (все страницы)
- Лого + "Smart Analyze / Sales Analytics Dashboard"
- Навигация: **Главная** | **Менеджеры** | **Библиотека паттернов**
- Справа: иконка юзера + название кабинета ("test cabinet" ▾)

### 1.2 Фильтр периода
- Переключатель: День | Неделя | Месяц | Квартал
- Влияет на ВСЕ данные на странице

### 1.3 Воронка продаж
- Горизонтальные карточки этапов в ряд
- Каждый этап:
  - Название этапа
  - ⏱ Время на этапе (X.X дн)
  - Конверсия % (зелёный/красный/жёлтый + стрелка →)
  - Прогресс-бар снизу
- **⚠ Восклицательный знак** на проблемных этапах (из расшифровки: "помеченные те этапы, на которые нужно обратить внимание")

### 1.4 Успех / Провал
- Две карточки рядом:
  - **УСПЕХ** (зелёный): иконка ✓, "Сделок закрыто: 22", "Общая сумма: 7 966 905,58 ₽"
  - **ПРОВАЛ** (красный): иконка ✕, "Сделок провалено: 30", "Общая сумма потерь: 10 863 962 ₽"

### 1.5 Потенциал роста выручки
- Большая цифра по центру: **18 830 868 ₽**
- Три метрики: Получено (ⓘ) | Потеряно (ⓘ) | Потери % (ⓘ)
- **ⓘ тултипы** с объяснением как считается каждая метрика
- CTA-блок: "Устранив провалы, отдел может увеличить выручку на 10 863 962 ₽"

### 1.6 Ключевые метрики
- 4 карточки: Конверсия | Ср. Чек | Ср. Время | Talk Ratio
- Каждая с **ⓘ тултипом** ("Здесь показывается каждая метрика, как она считается" — из расшифровки)
- Значение + тренд (↑/↓ за период)

### 1.7 График динамики
- Линейный график конверсии по времени
- Период-зависимый (из расшифровки: "можете видеть конверсию и динамику")

### 1.8 Рейтинг менеджеров
- Таблица: Менеджер | Сделок | Успешных | Конверсия | Ср. Чек | Ср. Время успешной сделки | Talk Ratio
- **Медали** у топ-менеджеров (🥇🥈🥉)
- Конверсия: цветная (зелёный/жёлтый/красный) + иконка тренда (↗/↘/→)
- **Клик по строке → переход в drill-down менеджера**

### 1.9 AI-инсайты по отделу (АККОРДЕОНЫ — КРИТИЧНО!)
- Два блока рядом:
  - **✓ Что работает лучше всего** (зелёный заголовок, иконка галочки в кружке)
  - **⚠ Что приводит к провалу** (красный заголовок, иконка треугольник с !)

- **Внутри каждого блока — аккордеоны**, раскрываются по клику (▾/▴)

- **Содержимое раскрытого аккордеона (ПОЛНАЯ СТРУКТУРА!):**
  1. **Заголовок паттерна** (uppercase, bold): "ГИБКОЕ КОНФИГУРИРОВАНИЕ ПРЕДЛОЖЕНИЯ ПОД БЮДЖЕТ КЛИЕНТА"
  2. **Краткое описание** (1-2 предложения): "Менеджер почти никогда не отказывает по цене напрямую, а подстраивает состав, тираж и конфигурацию заказа под бюджет клиента, сохраняя маржу без прямых скидок."
  3. **"Подробное описание:"** (красный заголовок) — развёрнутый AI-текст (2-3 абзаца):
     - Суть поведения
     - Конкретные приёмы менеджера (MOQ, технология нанесения, оплата на части, альтернативные ткани)
     - Почему это работает с точки зрения клиента
  4. **"Список сделок где встречается:"** — кликабельные chip-бейджи с номерами: `#90001` `#90003` `#90005` `#90008` `#90012` `#90015` `#90019`
     - Клик по номеру → **переход на страницу сделки**
  5. **"Список менеджеров:"** — chip-бейджи с именами: `Анна Петрова`
  6. **"Список цитат:"** — конкретные фразы из переписок:
     - "Могу отправить Вам бесплатный образец, чтобы вы оценили качество ткани (90001)"
     - "Дешевле будет если увеличите тираж до 50 единиц — выходит минус 15% от..." 
     - Каждая цитата с номером сделки в скобках

## Экран 3: Менеджер (drill-down)

### 3.1 Шапка
- Аватар + Имя + Роль
- Статус-pill (Отлично/На карандаше/Критично)

### 3.2 Метрики менеджера
- 5 карточек: Сделок | Успешных | Конверсия | Ср. Чек | Talk Ratio
- Каждая с ⓘ тултипом

### 3.3 График динамики менеджера
- Конверсия + время ответа по времени
- "Время реакции на каждого лида" (из расшифровки)

### 3.4 Успешные сделки (секция)
- Заголовок: "✓ Успешные сделки"
- Для каждой сделки — карточка:
  - Номер сделки + название + сумма + длительность
  - **"Что работает лучше всего"** — AI-текст: конкретные рекомендации
  - Из расшифровки: "не вода водениста, а конкретная рекомендация на сделках"
  - "менеджер почти никогда не отказывает по ценообразованию напрямую, а подстраивает состав тиражка заказа"
  - **"Выявленный паттерн успеха"** — какой паттерн сработал
  - **Ссылка** "провалиться в конкретную сделку" → переход на страницу сделки

### 3.5 Неуспешные сделки (секция)
- Заголовок: "✕ Неуспешные сделки" (для проблемных менеджеров)
- Карточка с анти-паттерном:
  - "клиент задаёт прямой вопрос про цену, получает встречные вопросы, каталог или общие фразы"
  - "Это основной анти-паттерн менеджера"
  - **Список цитат** — конкретные фразы менеджера, которые ведут к провалу
  - Из расшифровки: "список цитат, прям конкретные фразы, которые она делает"

## Глобальные UI-паттерны

### Тултипы ⓘ
- На КАЖДОЙ метрике есть иконка ⓘ
- При наведении/клике — попап "Как считается эта метрика"
- Из расшифровки: "Здесь показывается каждая метрика, как она считается"

### Навигационные связи (drill-down)
```
Главная → Рейтинг → клик менеджера → Менеджер drill-down
Менеджер → Успешная сделка → клик "провалиться" → Страница сделки
Менеджер → Паттерн → клик сделки (#90003) → Страница сделки
Главная → AI-инсайт → раскрыть → клик #90001 → Страница сделки
Библиотека → Паттерн → список сделок → клик → Страница сделки
```

### Аккордеоны (используются везде)
1. **Главная → AI-инсайты** — раскрывают полную структуру (описание + сделки + менеджеры + цитаты)
2. **Страница сделки → Дерево этапов** — раскрывают переписку каждого этапа
3. **Библиотека паттернов → Подробное описание** — раскрывает полный AI-текст

### Chip-бейджи
- Номера сделок: `#90001` `#90003` — кликабельные, ведут на страницу сделки
- Имена менеджеров: `Анна Петрова` — кликабельные, ведут на drill-down

### Цитаты
- Моноширинный или выделенный блок
- Текст цитаты + номер сделки в скобках
- "Могу отправить Вам бесплатный образец, чтобы вы оценили качество ткани (90001)"
